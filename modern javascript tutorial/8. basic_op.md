# 기본 연산자와 수학

자바스크립트의 기본 연산자는 다음과 같이 구분된다.

- unary op
  - negation
  - increament, decreament
    - postfix prefix의 차이
- binary op
  - 기본적인 사칙연산
  - power op (\*\*)
  - mouldar op (%)
  - assign op (=)
    - rvalue를 우선 평가
- ternary op
  - conditional ops

## 비트 연산자

- &
- |
- ^
- ~
- \<\<
- \>\>
- \>\>\>

## 쉼표 연산자

여러 표현식을 코드 한 줄에서 평가할 수 있게 한다. 반환값은 맨 마지막의 평가 결과이다.
